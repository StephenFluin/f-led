# Brightness Configuration Feature

## Overview

The ScoreBox now supports configurable LED brightness, allowing you to adjust the intensity of the score display from 0-100%.

## Brightness Scaling

The brightness feature uses a simple percentage scaling 0 - 100.

## Future Enhancements

Potential features to add:

- [ ] Separate brightness for home team vs opponent
